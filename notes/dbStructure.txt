Users
  _id: mongo id
  id: google id
  name: profile name
  pfp: link to profile pic
  locale: iso 2

Comments
  _id: mongo id
  parent: null or an _id of another comment
  video: youtube id
  content: text content
  likes: [List of user ids that liked the comment]
  dislikes: [List of users that disliked the comment]
  edited: has been edited or not
  author: [User id]
  time: unix timestamp (if edited, this is overwritten)

  For effiency (so I don't need to look up each user comment):
  username: authors username
  pfp: authors pfp

When a comment is saved or edited, it finds mentions (@Blah Blah) and sends them an email that they have been mentioned
